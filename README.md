# Système de Gestion de Bibliothèque

Projet Python réalisé dans le cadre du cours.

## Fonctionnalités
- **Partie 1** : gestion sans poo (partie1.py)
- **Partie 2** : gestion avec poo (main.py)

## Types de livres supportés
- Livre classique
- Livre Audio
- Bande Dessinée
- Ebook

## Lancer le projet
```bash
python partie1.py 
python main.py      
```

## Structure des fichiers
- `partie1.py` — Version sans poo
- `livre.py` — Classes Livre et sous-classes
- `utilisateur.py` — Classe Utilisateur
- `bibliotheque.py` — Classe Bibliothèque
- `main.py` — Programme principal
